# unlabeled_learning

### Installation

To install the package:

```
git clone <this repo> <a directory>
cd <that exact directory>
pip install cython
pip install -e .
```

[docs for `-e` option](https://pip.pypa.io/en/stable/reference/pip_install/#install-editable)


### Updating

```
<update repository>
python setup.py build_ext --inplace
```
